## This is the Phase 1 of the project and the whole project is divided into multiple phases (3)
### The goal for this Phase
Making the frontend that will later talk to our database and backend, use express.js for navigation (routing) and for now we will use the hardcoded data to ship the MVP of this whole project

At the end of this phase the site should be able to

1. Search a data locally
2. Filter the list for specific data
3. Clean and responsive UI
4. Show Cards of Data
5. Separate details page opens for each card (use Express routing)
